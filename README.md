# PROJECT - ENTOFU

개발기간 : 2022.05 ~ 2023.3 



### 프로젝트 소개
- - -

영어 학습에 어려움을 겪는 사람들이 편리하게 공부할 수 있도록 
하며 전반적인 영어 실력을 향상시키는데 도움을 주는 어플

### 환경

### 기능
- - -
- 앱에서 스크랩, 번역, 듣기, 필기, 단어장, 단어 발음 테스트 등의 학습 기능을 제공
- 매일 업로드 되는 영어 기사를 선택하고 쉬운 모션으로 학습
- 이미지, URL로 영어 지문을 앱에 업로드하여 개인 교재로 활용
- 다양한 형식의 랜덤 단어 퀴즈를 이용
